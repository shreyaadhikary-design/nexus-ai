# ⚡ NEXUS AI 
> "Where human curiosity meets machine intelligence."
Nexus AI is a high-speed, minimalist conversational assistant. Built with **Python** and powered by the **Llama 3** engine via **Groq**, it is designed for speed, privacy, and a clean user experience.
---
### 🚀 Key Features
* Instant Intelligence: Powered by Groq’s ultra-low latency Llama-3 API.
* Adaptive Memory: Remembers the context of your conversation in real-time.
* Clean UI: A distraction-free interface built with Streamlit.

### 🛠️ The Tech Stack
* Language: Python 3.10+
* Interface: Streamlit
* Brain: Llama-3-70B
